动态规划学习笔记
发布于 2019-05-18   更新于 2019-05-27 动态规划

动态规划基本概念

动态规划（又称 $DP$，$Dynamic$ $Programming$）是一种将问题分阶段求解的算法。

其能够通过利用先前阶段的计算结果快速得到后续阶段的结果，大大减少不必要的重复计算。

状态的设计是解决动态规划问题的关键。

设计状态时，要考虑到状态是否能够转移、转移的效率、以及求出所有所需状态的总时间复杂度。

状态的规模以及状态转移方程的实现效率共同决定了动态规划的效率。

线性DP

线性 $DP$ 是在线性序列上进行 $DP$ 的一类问题。

一些线性 $DP$ 其实等价于递推。

例题【洛谷P1020 导弹拦截】

$100$ 分

第一问是求其最长不升子序列，第二问是求其最长上升子序列，本质相同，以第二问为例。

令 $f[i]$ 表示以 $i$ 结尾的最长上升子序列。

则 $f[i]$ = $max(f[j])$ $+$ $1$ ，其中 $j < i$, $a[j] < a[i]$ 。

对于每个 $i$ 暴力枚举 $j 即可。

答案为所有 $f$ 的最大值。

时间复杂度 $O(n^2)$。

$200$ 分

考虑优化上述转移过程。

对于每个 $i$ ，需要查询最大的 $f[j]$ ，满足 $a[j]$ $<$ $a[i]$ 。

$f[j]$ 的值域为 $[1, n − 1]$ 。

我们可以记录 $d[x]$ 表示 $f$ 值为 $x$ 的最小 $a[j]$ 。

那么显然 $d$ 值是单调不下降的（反证法，若存在 $d[k+1] < d[k]$ ，则 $d[k+1]$所对应的长度为 $k + 1$ 的上升子序列的第 $k$ 个数 < $d[k+1] < d[k]$ ，即 $d[k]$应被更新）。

那么我们可以二分，找到最大的满足限制的 $x$ ，则 $f[i]$ $= x + 1$ 。

这样时间复杂度就被优化到了 $O(n log n)$。

区间DP

区间 $DP$ 的思想是由小区间的状态合并得到大区间的状态。

例题：【洛谷P4290 [HAOI2008]玩具取名】

令 $f[l][r][c]$ 表示名字的 $[l,r]$ 区间是否可以由 $c$ 字母变形而来。

目标是求 $f[1][n][c]$ ，其中 $c ∈ {W, I, N, G}$ 。

按区间长度从小到大的顺序进行 $dp$。对于 $f[l][r][c]$ ，枚举第一个字母（也就是 $c$ ）变形成的两个字母是什么，及分别对应 $[l,r]$ 中的哪些位置，即可利用之前的 $dp$ 信息求出。

树形DP

例题：【洛谷P1352 没有上司的舞会】

令 $dp[i][1]$ 表示以 $i$ 为根的子树中选取任意多互不相邻的点的最大权值。

令 $dp[i][0]$ 表示以 $i$ 为根的子树中选取任意多互不相邻的点，且 $i$ 点本身不选的最大权值。

则 $dp[i][0] =∑ dp[k][1]$ ，其中 $k$ 为 $i$ 的儿子。及 $dp[i][1] = max(dp[i][0],∑ dp[k][0] + v[i])$ ，其中 k 为 i 的儿子。

假设根节点为 $x$ ，则答案为 $dp[x][1]$ 。

背包DP

一类形如“有容量为 $S$ 的背包，有若干体积 $v[i]$ ，价值 $w[i]$ 的物品，求最大能拿的价值”的问题。

有很多变式，如 $01$ 背包、完全背包、多重背包、有依赖的背包、分组背包等等。

推荐阅读《背包九讲》。
